Features

Traffic Agent uses Deep Learning Algorithms to predict the traffic signs. It implements various deep learning libraries to build Robust and Efficient Convolutional Neural Network which can provide you Fast and Accurate Results.

Technology Stack of the Traffic Agent:

  • Python
  • HTML, CSS, JavaScript
  • Bootstrap
  • Flask
  • TensorFlow, Keras
  • Pandas, Numpy, Scipy
  • To be Continued...

Privacy

Traffic Agent respects your privacy. Your uploaded Images get deleted after an hour. Even It doesn't have any annoying ads so that you can predict traffic signs without any kind of disturbance or hesitation.

Privacy-First Features:

  • No Data Collection
  • No Cookies stored in your browser
  • Automatic deletion of your Images
  • No Ads
